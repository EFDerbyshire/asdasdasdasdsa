Probably the biggest repo I've done alone. This repo contains a fair quantity of edits for a RPG gamemode for the physics sandbox Garry's Mod that I was making for a community server of the same name, also ran by yours truly. This was an entirely personal project, though I did receive input from others, every commit to this repo with the exception of some plugins was authored solely by me. That's not to say it's impressive, mind you. Because Garry's Mod uses Lua (not exactly the most difficult language in the world) every change authored was hardly that difficult an enterprise.

Garry's Mod does, however, use this wonderful thing called Derma. The best way to describe Derma is as the GUI language from hell. On top of being counterintuitive, difficult to understand, and honestly just difficult outright, any attempts I made to use it would flop after the first panel or two.

Honestly, if you want to know how horrible Derma is to understand and get working, just head over to the developer docs. I'm sure you'll have a great time there.

I've since archived this one, but it remains on GitHub here.
